q
- binomial coefficient calculator













































































This calculator is developed by yodai suzuki.It enables to calculate the value of q-binomial coefficient.
























Q-binomial coefficient is also called "Gaussian binomial coefficient".It is q-analogue of binomial coefficient.
























This web page provides the value of q-binomial coefficient using the equality "nCk_q=q^(n-k)*(n-1)C(k-1)_q+nC(k-1)"
